# PCB-Badge
Intro project for Fallout, cool badge in the shape of lightning bolt (Because ZeusyBoy). Has NFC so people can get my github.

## BOM
| Item                            | Quantity |
| ------------------------------- | -------- |

| Resistor 47Ω 0603               | 1        |
